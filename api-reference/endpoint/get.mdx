## Endpoint

```
GET /v2/detectors/{id}/measurements
```

Retrieve temporal field measurements from a detector.

<Info>
  Measurements are returned in chronological order. Use pagination for large datasets.
</Info>

## Request Parameters

### Path Parameters

- `id` (string, required) - Detector device ID (e.g., `tfd_500_01`)

### Query Parameters

- `start_time` (string, optional) - ISO 8601 timestamp for start of range
- `end_time` (string, optional) - ISO 8601 timestamp for end of range
- `limit` (integer, optional) - Maximum number of measurements to return (default: 100, max: 1000)
- `cursor` (string, optional) - Pagination cursor from previous response
