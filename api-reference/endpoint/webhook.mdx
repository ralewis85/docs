## Endpoint

```
POST /v2/webhooks
```

Configure webhook endpoints to receive real-time notifications about resonator events, field anomalies, and safety alerts.

## Event Types

- `resonator.engaged` - Resonator successfully engaged
- `resonator.stopped` - Resonator stopped
- `coherence.warning` - Field coherence dropped below threshold
- `coherence.critical` - Critical coherence level reached
- `anomaly.detected` - Field anomaly detected
- `safety.emergency_stop` - Emergency stop triggered

<Warning>
  Webhook endpoints must respond within 3 seconds to avoid timeout.
</Warning>

## Request Parameters

### Body Parameters

```json
{
  "url": "https://your-server.com/webhooks/tre",
  "events": ["resonator.engaged", "coherence.warning"],
  "secret": "whsec_...",
  "enabled": true
}
```

- `url` (string, required) - HTTPS URL to receive webhook events
- `events` (array, required) - List of event types to subscribe to
- `secret` (string, optional) - Webhook signing secret for verification
- `enabled` (boolean, optional) - Enable/disable webhook, default: true

## Response

### Success Response (201 Created)

```json
{
  "webhook_id": "wh_abc123def456",
  "url": "https://your-server.com/webhooks/tre",
  "events": ["resonator.engaged", "coherence.warning"],
  "enabled": true,
  "created_at": "2024-11-03T15:30:00Z"
}
```

## Webhook Payload Example

```json
{
  "event_id": "evt_xyz789",
  "event_type": "coherence.warning",
  "timestamp": "2024-11-03T15:30:15Z",
  "data": {
    "resonator_id": "tres_9000_01",
    "coherence": 0.82,
    "threshold": 0.85
  }
}
```
